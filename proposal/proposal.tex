\begin{abstract}
  Recently, the game Wordle, by Josh Wardle, has become quite popular. It is a game in which the player gets 6
  guesses to determine a random 5 letter word. We propose development of a reinforcement learning model that will
  be capable of playing the game.
\end{abstract}
\section{Introduction}
Applying reinforcement learning techniques to solve games is a popular project topic . Reinforcement
learning has also been used to solve Scrabble --- a game fundamentally similar to Wordle.

There are existing automated solutions to Wordle, the most prominent of which being the 3 blue 1 brown solution.
There is also a Reinforcement Learning version that exists that uses DQN and A2C.

We intend to develop an OpenAI gym environment to train the agent in, and then create multiple agents to compare
performance of different algorithms on this problem. Finally, just for fun, we will attempt to create a wrapper
that makes the agent able to play the acutal Wordle game on the website, as well as clones of the game.
